# Complaint Management System (MySQL)

This project is a Complaint Management System database created using MySQL.  
It helps to manage user complaints, assign them to admins, track complaint status, and store feedback.

## Database Tables

1. Users
2. Admins
3. Complaints Record
4. Complaint Categories
5. Complaint Priority
6. Complaint Status
7. Complaint Assignment
8. Assistance
9. Management
10.Notifications
11.Escalations
12.Feedback
13.Attachments
14.Roles
15.Resolutions Details

## Features

- Users can register and raise complaints
- Complaints are assigned to admins
- Complaint priority and status tracking
- Complaint escalation system
- Feedback collection after resolution
- Notification management

## Technologies Used

- MySQL
- SQL Queries
- Joins
- Aggregate Functions
- Database Relationships
